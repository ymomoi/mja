# mja : 手動麻雀点数表示ボード MahJongg Assistant board

点数表示のない麻雀卓でプレイするときに点棒のやりとりをせずに遊ぶために作った、ニッチなプログラムです。
スマホやタブレットのブラウザで表示して使うため、固定サイズ/幅のシンプルなレイアウトをしています。
